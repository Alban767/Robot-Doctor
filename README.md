#DOC247 version 1.0
It is a project about the first version DOC247. Although the latest version of DOC247 looks completely different because of being more advanced with actaul diagnosis capabalities using both visual, audio, and text data input from a patient including data corrected automatically from the DOC247 hardware equiped with actaul sensors, diagnising patients and dispensing actual medicine, among other features, this version was the inspiration. 
DOC247 is a robot doctor and works as a professional medical assistant.
